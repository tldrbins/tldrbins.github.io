### General

<div>

```bash
# Open a db
sqlite3 sql.db
```

```bash
# Show tables
.tables
```

```bash
# Show schema of a table
.schema <TABLE_NAME>
```

```bash
# Show column headers
.headers on
```

```bash
# Dump data
select * from <TABLE_NAME>;
```
