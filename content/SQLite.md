### General

```bash
# Open a db
sqlite3 sql.db
```

```bash
# Show tables
.tables
```

```bash
# Show schema of a table
.schema table_name
```

```bash
# Show column headers
.headers on
```

```bash
# Dump data
select * from table_name;
```
