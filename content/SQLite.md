### General

```console
# Open a db
sqlite3 <DB_FILE>
```

```console
# Show tables
.tables
```

```console
# Show schema of a table
.schema <TABLE_NAME>
```

```console
# Show column headers
.headers on
```

```console
# Dump data
select * from <TABLE_NAME>;
```
