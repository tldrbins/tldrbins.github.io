### searchsploit

```bash
# Search service with specific version
# For example: samba 3.0
searchsploit samba 3.0

# Read the exploit
searchsploit -x unix/remote/16320.rb

# Copy the exploit
searchsploit -m unix/remote/16320.rb
```
