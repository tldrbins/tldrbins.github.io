Using Telnet

# Connect
telnet 127.0.0.1 11211

Using memcached-cli

# Install
sudo npm install -g memcached-cli

# Connect (With Creds)
memcached-cli username:password@10.10.11.10:11211

# Info about slabs

# Show slab info (e.g. slab 1 and display all keys)
stats cachedump 1 0

# Dump info
