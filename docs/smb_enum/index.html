Mount SMB Share

sudo mount -t cifs //<TARGET>/<SHARE> /mnt

Sample Output:
TO-DO
sudo mount -t cifs -o ro,user='<USER>',password='<PASSWORD>' //<TARGET>/<SHARE> /mnt

Sample Output:
$ sudo mount -t cifs -o ro,user='localadmin',password='Secret123' //10.10.11.102/Shared /mnt

$ ls /mnt
