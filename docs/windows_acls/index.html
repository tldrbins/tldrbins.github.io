Window ACLs (Access Control Lists)

Check ACLs

# Check a file/directory
icacls <PATH>

# Check all files and directories
icacls * /C

Grant user full control

# cmd
cmd.exe /c cacls <FILE> /E /G <USER>:F

# powershell
icacls <FILE> /grant <USER>:F

Change owner (WO)

# cmd
cmd.exe /c takeown /F <FILE>
