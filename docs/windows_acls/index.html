Window ACLs (Access Control Lists)

Check ACLs

# Check a file/directory
icacls "C:\Users\User"

# Check all files and directories
icacls * /C

Grant user full control

# cmd
cmd.exe /c cacls file_name /E /G user:F

# powershell
icacls file_name /grant user:F

Change owner (WO)

# cmd
cmd.exe /c takeown /F file_name
