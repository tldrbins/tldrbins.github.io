Create a Wordlist

# For example, 'Test' with 4 digits and a symbol
hashcat --stdout -a 3 Test?d?d?d?d?s > passwords.txt

Sample Output:
$ hashcat --stdout -a 3 Test?d?d?d?d?s > passwords.txt
$ head passwords.txt 
Test1999*
Test2312.
Test5678*
Test0000.
Test9000.
Test8888*
Test3000.
Test4523.
Test7778*
Test6999*


+------------------------------------------+
|  ? | Charset                             |
| ===+=========                            |
|  l | abcdefghijklmnopqrstuvwxyz [a-z]    |
|  u | ABCDEFGHIJKLMNOPQRSTUVWXYZ [A-Z]    |
|  d | 0123456789                 [0-9]    |
|  h | 0123456789abcdef           [0-9a-f] |
|  H | 0123456789ABCDEF           [0-9A-F] |
|  s |  !"#$%&'()*+,-./:;<=>?@[\]^_`{|}~   |
|  a | ?l?u?d?s                            |
|  b | 0x00 - 0xff                         |
+-----------------------------------------+
