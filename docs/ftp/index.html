Config Location

/etc/vsftpd.conf

Basic

# Connect
ftp 10.10.11.10

+-------------------+
|Anonymous login    |
+-------------------+
|Username: anonymous|
|Password: (empty)  |
+-------------------+

# List
ls

# Switch to binary transfer mode
bin

# Download
get filename

# Upload
put filename

# Connect over tunnel
passive

# Exit
quit

Recusive download

# anonymous login
wget -r ftp://anonymous:@10.10.11.10

# with creds
wget --user username --password password -m ftp://10.10.11.11

Note: Always check what’s in there first
