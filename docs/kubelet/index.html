Tools

kubeletctl

Basic Commands

# List all the pods on the node
kubeletctl pods -s 10.10.11.10

# List running pods
kubeletctl runningpods -s 10.10.11.10

# Exec command
kubeletctl exec "/bin/bash" -p pod_name -c container_name -s 10.10.11.10
